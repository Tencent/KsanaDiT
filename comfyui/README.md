

KsanaDiT 支持作为 ComfyUI 的自定义节点使用，提供可视化工作流体验：

```bash
# 1. 进入 ComfyUI 的 custom_nodes 目录
cd /path/to/ComfyUI/custom_nodes

# 2. 克隆 KsanaDiT 仓库
git clone https://github.com/Tencent/KsanaDiT.git

# 3. 进入 KsanaDiT 目录并安装依赖
cd KsanaDiT
./scripts/install_dev.sh
```

安装完成后，重启 ComfyUI 即可在节点列表中看到 KsanaDiT 相关节点。更多使用说明请参考[ComfyUI](https://github.com/Comfy-Org/ComfyUI)
